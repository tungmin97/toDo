<h1 align="center">ToDo List 📑</h1>

#### Task list application

A small application developed in React Native with React Native CLI using a minimalist design. User can create, delete tasks and mark them as complete.

## Install

```sh
yarn
```

## Usage

```sh
yarn android
```
